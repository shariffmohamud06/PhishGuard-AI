# 🛡️ PhishGuard AI

An AI-powered phishing detection agent built during Microsoft AI Skills Fest 2026.

## 🎯 What It Does
PhishGuard AI analyses suspicious emails and URLs to detect phishing attempts using 
Azure AI. It explains why content is dangerous and suggests what action to take.

## 🛠️ Built With
- Python
- Azure AI Services
- Microsoft Azure
- GitHub Copilot

## 🚀 Features
- Detects phishing emails using AI classification
- Explains why content is flagged as suspicious
- Built as part of the Agents League Hackathon 2026
